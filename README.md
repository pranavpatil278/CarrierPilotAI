#  🚀 CarrierPilotAI

CarrierPilotAI is a full-stack career assistant web application that helps users analyze resumes, calculate ATS scores, and generate personalized resume improvement suggestions using AI.

## Features

- Upload and analyze resumes
- Get ATS compatibility scoring
- Receive suggestions to improve resume quality
- Built with React frontend and FastAPI backend

## Tech Stack

- Frontend: React, Vite, Tailwind CSS
- Backend: FastAPI, Uvicorn, Python
- Data: File upload and AI resume analysis services

## Project Structure

- `Backend/` - FastAPI backend server
- `Frontend/` - React frontend application

## Setup

1. Clone the repository
2. Install backend dependencies:
   ```bash
   cd Backend
   pip install -r requirements.txt
   ```
3. Install frontend dependencies:
   ```bash
   cd ../Frontend
   npm install
   ```

## Run Locally

- Start backend:
  ```bash
  cd Backend
  uvicorn main:app --reload
  ```

- Start frontend:
  ```bash
  cd Frontend
  npm run dev
  ```

## Notes

- Add any required environment variables to `Backend/.env`
- Update API endpoints in frontend if backend URL changes
